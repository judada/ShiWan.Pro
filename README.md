### 什么是试玩？ 
APP厂商花钱给ASO平台，通过刷榜的形式来提高app在AppStore的整体表现（例如排名）。普通用户通过平台接受任务，根据指定关键字去AppStore下载指定app体验。
>安装试玩平台-->接任务-->根据平台提示到指定应用商店下载指定app-->根据要求进行体验-->体验结束，返回平台领取奖励
