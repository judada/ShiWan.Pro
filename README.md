### 什么是试玩 
APP厂商花钱给ASO平台，通过刷榜的形式来提高app在AppStore的整体表现（例如排名）。用户通过平台接受任务，根据指定关键字去AppStore下载指定app体验，然后回任务平台领取奖励。

### 试玩准备
型号7及以上的苹果手机、appleid账号、实名信息(手机号码、微信、支付宝)。<br/>
二手苹果机子是个不错的选择，国行7 二手的价格一般在600-900左右
